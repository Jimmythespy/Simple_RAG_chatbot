# Simple chatbot using Langchain & FastAPI and chatGPT
Description: A simple chatbot application using RAG and ChatGPT


- **To start the server run**: uvicorn main:app --reload
- **To use the chatbot**: go to http://127.0.0.1:8000/query_ui
- **To add more document and website**: go to http://127.0.0.1:8000/uploadfile_ui
